# Changelog 2026-03

## 2026-03-20 — Mobilní flowchart + odemykání scénářů + donate

### Flowchart na mobilu (hlavní téma session)
- **AppHeader: sticky → fixed** — sticky nefungoval s overflow:hidden, fixed top-0 je vždy přilepený k viewportu
- **FlowchartPage: fixed layout** — `fixed inset-0 top-16` místo calc výšky, body overflow:hidden při mount
- **svg-pan-zoom → panzoom** — svg-pan-zoom měl nefunkční touch na mobilu, nahrazen panzoom (stejný jako MapPage)
- **Tap detekce** — samostatný touchstart/touchend listener (< 15px pohyb, < 300ms) vedle panzoom
- **ViewBox** — portrait zůstává 420×1080 (plný, včetně vedlejších scénářů)
- **Přehled v mobilním menu** — odstraněn redirect /scenare, vede na /prehled
- **Duplikátní legenda** — odstraněna spodní, zůstává jen v toolbaru
- **App.vue** — main content pt-18 (kompenzace fixed headeru)

### Odemykání scénářů
- **scenarioStore.unlockScenario()** — nová funkce, uloží stav AVAILABLE
- **computedStatuses** — Pass 1 respektuje manuálně uložený AVAILABLE stav
- **ScenarioDetail** — modré tlačítko "Odemknout scénář" pro LOCKED/BLOCKED
- **ScenariosPage modál** — přidány akční tlačítka (odemknout/dokončit/pokus/reset)
- **Reaktivní modál** — selectedScenario je computed (ne statický snapshot), tlačítka se mění po akci
